# telefrag-ui
UI компоненты для проекта [The Daily Telefrag](https://dailytelefrag.ru)
В прошлом пакет [@alxgrn/react-form](https://www.npmjs.com/package/@alxgrn/react-form)

## Установка

```
npm install @alxgrn/telefrag-ui
```
