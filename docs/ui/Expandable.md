# Expandable
Компонент для отображения блочного элемента, который необходимо ограничить по высоте, с возможностью полного раскрытия.

```jsx
return(
    <Expandable>
        <p>Каждый охотник желает знать, где сидит фазан.</p>
    </Expandable>
);
```

## Свойства и методы
|Prop name|Type|Default|Description|
|---------|----|-------|-----------|
|maxHeight?|number|300|Максимальная высота свернутого блока|
|aspectRatio?|`1x1`\|`16x9`|`1x1`|Соотношение сторон свернутого блока|
|expand?|string|`Показать целиком...`|Текст для отображения в ссылке раскрытия блока|
|collapse?|string|`Свернуть...`|Текст для отображения в ссылке сворачивания блока|
|isCollapsable?|boolean|true|Флаг того что блок можно сворачивать после раскрытия|
|noShadow?|boolean|false|Флаг того что не нужно показывать затенение нижнего края у свернутого блока|

Можно указывать один из параметров: `maxHeight` или `aspectRatio`. Нельзя указать сразу оба, но можно не указывать ни одного, в этом случае будет использоваться `aspectRatio=1x1`.

Если изначально размер блока меньше чем `maxHeight` или меньше блока размерами вычисляемыми через указание `aspectRatio`, то ссылка разворачивания блока не показывается.

Блок корректно обрабатывает изменение размера внутреннего компонента.

У блока есть минимальная высота в `50px` (высота затенения нижнего края свернутого блока), меньше которой в свернутом состоянии он быть не может.
